What is indi - TA?

indi - TA is an acronym for
integrated n-dimensional interface - Technical Analysis.

What does that mean?

It's technical analysis that analyze the chart as n-dimensional vector space. It's not compatible with some of the axioms of abstract algebra.

What model for price does indi - TA use?

indi - TA uses a quantum model for price that is based on the mechanics of the auction process.

How does it work?

Noether's theorem can be used to find relationships between continuous symmetries and conserved quantities. These relationships can be charted with n-dimensional vector space. Information theory can be used to compress data from n-dimensional vector space to form a discretionary trading system.
