Anchored Notes

Section List Text

This is how JC setup anchored notes in the object tree.

If it says no folder, then use a Separator template for the anchored note. No folder is needed.

If it has a folder name, then use a placeholder template in that folder. Make the anchored note first then create the folder.

Technical Analysis
Folder Name Anchor name Note Text
no folder ** TECHNICAL ANALYSIS ** Technical Analysis
(* = live, / = inactive)
Tools: Rectangle, Price Range, Trend Line

Description: Make folders to organize the object tree.
Put new stuff in this section if there's no time to sort it.

k = key level
sr, sg = spike red, spike green
@ = alert
Example: k1, k2, k3, sr, sg, @=k1, @=p9s
